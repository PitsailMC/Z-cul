A BFRC fork with additional improvements and a guaranteed FPS increase


Compatibility

 
Due to the use of GPU instantiation to assist computation,entity culling functionality cannot run on graphics card devices that do not support OpenGL 3.3.
If your device can run the Sodium, it can also support instantiation entity culling functionality. This mod utilizes a portion of GPU computation to alleviate CPU culling and chunk compilation overhead.
If your GPU doesn't have additional computational power to render culling textures,
it may slightly reduce the maximum frame rate in open vistas, typically below 5%.
